# operato-fnb
Operato F&B brand project for windows 

## Command Guide

### Application Run

```cmd
start.bat
```

### Database Migration

```cmd
migrate.bat
```

### Dependency Module Upgrade
```cmd
upgrade.bat
```

### Service Registration
on x64 environment
```cmd
nssm_install_x64.bat
```
on x86 environment
```cmd
nssm_install_x86.bat
```
