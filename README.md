# AIdoop-r Windows Installation
aidoop-r project for windows 

## Command Guide

**All commands should run with administrator's privilege.**


### Application Run

```cmd
start.bat
```

### Database Migration

```cmd
migrate.bat
```

### Dependency Module Upgrade
```cmd
upgrade.bat
```

### Service Registration
```cmd
nssm_install.bat
```
